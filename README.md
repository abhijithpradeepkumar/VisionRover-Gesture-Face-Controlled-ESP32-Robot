# VisionRover: Gesture & Face Controlled ESP32 Differential Drive Robot

VisionRover is a computer vision enabled mobile robot built using ESP32 and ESP32-CAM.

## Features

### Current

* ESP32 based differential drive control
* 4-wheel Johnson motor drive
* Dual L298N motor drivers
* Battery powered operation

### In Progress

* ESP32-CAM video streaming
* Face detection
* Gesture recognition
* Wi-Fi control interface

### Future

* Human following
* Face recognition
* Autonomous navigation
* Obstacle avoidance
* Remote monitoring dashboard

## Hardware

* ESP32 DevKit V1
* ESP32-CAM
* 2 × L298N Motor Drivers
* 4 × Johnson DC Motors
* 7.4V Li-ion Battery
* Voltage Regulator Module
* Ultrasonic Sensor
* Servo Motor
